# Creare custom GitHub Actions

_Crea la tua GitHub Action con JavaScript e automatizza attività specifiche del tuo lavoro._

## Benvenuto

- **Per chi è**: Studenti, sviluppatori, utenti GitHub, utenti nuovi a Git, manager e team.
- **Cosa imparerai**: Come creare GitHub Actions personalizzate basate su JavaScript, pacchettizzarle per la distribuzione e utilizzarle nei workflow.
- **Cosa costruirai**: Una GitHub Action basata su Javascript che recupera battute "dad jokes" casuali e le espone come output.
- **Prerequisiti**:

  - Dovresti avere familiarità con GitHub e GitHub Actions. Ti consigliamo di seguire prima l'esercizio [Hello GitHub Actions](https://github.com/skills/hello-github-actions).

- **Quanto tempo**: Questo esercizio richiede meno di 1 ora per essere completato.

In questo esercizio:

1. Inizializzerai un progetto JavaScript
2. Creerai file sorgente JavaScript per una action "Dad Jokes"
3. Pacchettizzerai la tua action in un file distribuibile
4. Creerai i metadati della action con `action.yml`
5. Creerai un workflow che utilizza la tua action personalizzata
6. Testerai la tua action attivandola con i commenti delle issue

### Come iniziare questo esercizio

1.  Fai clic con il tasto destro su **[Step 0](../../actions/workflows/0-start-exercise.yml)** e aprilo in una nuova scheda.
2.  Clicca il pulsante **Run workflow**.
3.  Seleziona il branch `main` e clicca il pulsante verde **Run workflow**.
4.  Attendi circa 20 secondi e aggiorna questa pagina.
